# 医療バンクシー

国民の医療リテラシー向上と医療費削減を目指す医療情報サイト

## プロジェクト概要

医療バンクシーは、診療ガイドラインに準拠した信頼できる医療情報を、分かりやすく提供するウェブサイトです。

### 2つの目的

1. **国民の医療リテラシーを高める**
   - 診療ガイドラインに準拠した信頼できる情報を提供
   - 適切な受診判断ができるようになる
   - 不要な心配を減らす

2. **医療費の軽減**
   - 個人の医療費負担を軽減
   - 国民が医療費削減の道筋があることを知る
   - 政治を動かし、持続可能な医療制度を実現

## セットアップ

### 必要なもの
- Webブラウザ
- （オプション）ローカル開発サーバー

### ローカルで実行

1. リポジトリをクローン
```bash
git clone https://github.com/your-username/medical-banksy.git
cd medical-banksy
```

2. ブラウザでindex.htmlを開く
   - 直接ファイルを開く、または
   - ローカルサーバーを起動:

```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsのhttp-serverを使う場合
npx http-server -p 8000
```

3. ブラウザで `http://localhost:8000` にアクセス

## コンテンツ戦略

### 0次予防から3次予防まで

**0次予防：自分を知る**
- 体質チェック、MBTI、自律神経の理解

**1次予防：病気を防ぐ**
- 食事、運動、睡眠、ストレス管理

**2次予防：早く気づく**
- セルフチェック、受診の目安、健康診断

**3次予防：上手に治す**
- 市販薬の選び方、適切な受診、リハビリ

### コンテンツタイプ

1. **症状別ガイド**
   - 風邪、頭痛、花粉症、腹痛など
   - 受診の目安、市販薬の選び方

2. **ケーススタディ**
   - 医療費削減の実例
   - 計算根拠を明示

3. **図解コンテンツ**
   - 30秒で要点をつかめる

4. **動画コンテンツ**
   - ショート動画（30秒〜1分）
   - 解説動画（3〜5分）

## ガイドライン準拠

### 参照ガイドライン（例）
- 日本アレルギー学会「鼻アレルギー診療ガイドライン」
- 日本頭痛学会「頭痛の診療ガイドライン」
- 厚生労働省「抗菌薬の適正使用」
- 日本うつ病学会「気分障害の治療ガイドライン」

### 表記に関する方針

**記載OK：**
- 一般的な症状の説明
- 公的ガイドラインの引用・要約
- 薬のカテゴリー名
- 受診の目安

**記載NG：**
- 診断行為
- 処方の指示
- 効果の断定
